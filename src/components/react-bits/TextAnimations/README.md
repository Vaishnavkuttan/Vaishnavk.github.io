# React Bits - Text Animations

Run these commands to add components to your project. 
Note: Since we are using a custom structure, you might need to move the files to `src/components/react-bits/TextAnimations/` after installing.

## List of Components

### Split Text
```bash
npx shadcn@latest add @react-bits/SplitText-JS-CSS
```
